# scala-manipulate-hbase
## 用scala来做操作HBase
[HBaseUtils.scala](https://github.com/user1508809/scala-manipulate-hbase/blob/master/src/main/scala/HBaseUtils.scala)
## HBaseUtils方法说明
1. 获取Admin`getAdmin` 
2. 获取表 `getTable` 
3. 创建表 `createTable` 
4. 删除表 `delete` 
5. 放数据到表中 `put` 
6. 从表中获取数据`get` 
7. 扫描表中的数据`scan`
